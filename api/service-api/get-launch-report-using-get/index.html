Export specified launch

GET 
/v1/:projectName/launch/:launchId/report

Only following formats are supported: pdf (by default), xls, html.

Request​

Path Parameters

    launchId int64required

    launchId

    projectName stringrequired

    projectName

Query Parameters

    view string

    Possible values: [pdf, xls, html]

    Default value: pdf
