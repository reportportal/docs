# Edit personal information

To navigate to the User Profile page, open menu at the bottom of the page and select "My profile" option.

The User Profile page consists of the following elements:

- Personal information and photo – JPEG, PNG, GIF file, with a file size limit no more than 1 MB and 300x500px.

- Assignments – a read-only list of your organizations and projects, along with the roles associated with them.

- API Keys - unique tokens that grant access to the ReportPortal REST API.

- Configuration Examples

## Edit name, email and password

You can edit the name, email and password fields.

:::note
The specified login at user
creation cannot be changed.
:::
To edit the full name and/or email, click on the pencil icon. The full
name and email fields will become available.

<MediaViewer src={require('./img/EditPersonalInfo2.png')} alt="Pencil icon" />

Full name may contain only Latin, numeric characters, symbols: hyphen, underscore, apostrophe, dot. The email should be unique in ReportPortal.

Make changes and click the "Submit" button.

<MediaViewer src={require('./img/EditPersonalInfo3.png')} alt="Edit name, email and password" />

<MediaViewer src="https://youtu.be/yVdo-C8kkkU" alt="Edit Personal Info in our test automation results dashboard" type="video" />

:::note
Personal information for users who were created in ReportPortal via auth providers (e.g. GitHub, SAML etc.) cannot be changed.
:::
## Change password

To change your password in ReportPortal, click on the "Change password" button
above the form and enter:

- Old password
- New password
- Confirm new password

Check the box "Show password" to verify the password entered is what you intended to enter.

Fill in these fields and click the "Submit" button under the form.

<MediaViewer src="https://youtu.be/WzcQsK-PzyM" alt="Change Password real time dashboard open source" type="video" />

:::note
Users who were created in ReportPortal via auth providers (e.g. GitHub, SAML etc.) cannot change the password in ReportPortal.
:::

## Language setting

In ReportPortal, you can personalize your experience by selecting a preferred interface language.

The language settings are available on the Profile page, where users may choose from six supported languages: English (United States), Ukrainian, Russian, Belarusian, Chinese, Spanish. Once selected, the chosen language will be applied across the entire application interface.
