# Projects

You can access the "Projects" page using sidebar from organization level.

On the "Projects" page Admin and Manager can see the list of projects in the organization. Member can only see the projects they are assigned to.

<MediaViewer src={require('./img/projects/Projects1.png')} alt="Projects page in our test automation reporting dashboard" />

The page provides next information:

* Organization name
* Projects – number of projects in the organization (available for Admin, Manager)
* Users – number of users in the organization (available for Admin, Manager)
* Project name – name of the projects in organization
* Teammates – number of users assigned to the project
* Launches – the number of reported launches per the project
* Last launch date – the date when last launch was reported

"Projects" page allows:

* to create a project
* to rename a project
* to invite users to a project
* to quickly assign/unassign yourself to/from project (available for Admin, Manager)
* to delete a project

In the header of the "Projects" page, the following controls are displayed:

* Search
* Filter
* "Create project" button

<MediaViewer src={require('./img/projects/Projects2.png')} alt="Controls in the header" />

The user can search for projects available to them.

## Filter

The user can filter the list of projects available to them by the following criteria:

* Last run date
* Launches
* Teammates

<MediaViewer src={require('./img/projects/Projects4.png')} alt="Filter the projects" />

## Create project

To create a project from the "Projects" page:

1. Log in as Admin/Manager/Member.
2. Navigate to organization level -> "Projects" page.
3. Click on the "Create project" button.
4. Enter project’s name.
5. In the modal window, click on "Create" button.

<MediaViewer src={require('./img/projects/Projects5.png')} alt="Create project" />

## Rename project

To rename a project from the "Projects" page:

1. Log in as Admin/Manager/Member.
2. Navigate to organization level -> "Projects" page.
3. Click the meatball menu on the project.
4. Choose "Rename" option.

<MediaViewer src={require('./img/projects/Projects6.png')} alt="Rename option" />

5. Enter new name.
6. In the modal window, click on "Rename" button.

<MediaViewer src={require('./img/projects/Projects7.png')} alt="Renamed project" />

## Invite user to project

To invite a user to the project from the "Projects" page:

1. Log in as Admin/Manager/Member.
2. Navigate to organization level -> "Projects" page.
3. Click the meatball menu on the project.
4. Choose "Invite user" option.

<MediaViewer src={require('./img/projects/Projects8.png')} alt="Invite user option" />

5. Enter email.
6. Check the "Can edit the Project" checkbox if needed.
7. Click on "Invite" button.

<MediaViewer src={require('./img/projects/Projects9.png')} alt="How to Invite user to project" />

To delete a project from the "Projects" page:
